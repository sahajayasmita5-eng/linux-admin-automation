# Linux Admin Automation Scripts

This repository contains simple Bash scripts created for automating routine Linux administration tasks.

## Script: Password Rotation

This script automates password refresh for a service account across multiple Linux servers using SSH.

### Features
- Reads server list from file
- Connects via SSH
- Updates user password
- Verifies password update status using chage -l

### Tech Used
- Bash
- SSH
- Linux
